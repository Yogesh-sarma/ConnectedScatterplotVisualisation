Pan and Zoom interface description

Panning -

User can pan the graph using these 4 keys - "Arrow Up", "Arrow Down", "Arrow left", "Arrow right"

Zooming -

User can zoom using these 2 keys - "+" to zoom in and "-" to zoom out

Diverging Colormaps -

Change in "Miles" is used to get the hue. Can be changed using dropdowns

Dataset -

The dataset I have taken is the accident data under different lighting conditions in North Carolina state since 1975. Data includes number of accidents per year when it is daylight, dusk, dawn, dark, and other light conditions

Source of data set : https://www.nhtsa.gov/file-downloads?p=nhtsa/downloads/FARS/2020/National/

The above data contains accident information of all the states. I have filtered the data to include only North Carolina state accidents data.
